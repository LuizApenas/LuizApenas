## Sobre

Atuo no desenvolvimento de aplicações full stack com foco em sistemas de negócio, produtos internos e fluxos operacionais reais.
Meus repositórios mostram uma trajetória que passa por CRM, e-commerce, gestão de caixa, automações em Python, mobile e projetos web publicados.

- Backend com Go, Node.js e APIs voltadas a regras de negócio.
- Frontend com React, Next.js, TypeScript e interfaces orientadas a uso real.
- Dados e integrações com PostgreSQL, Supabase, Firebase, GraphQL e automações.
- Infra aplicada ao produto com Docker, EasyPanel, Vercel e configuração de ambientes.
- Interesse constante em qualidade, testes, DX, padronização e manutenção de longo prazo.
