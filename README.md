## Sobre

Atuo no desenvolvimento de sistemas e ferramentas internas com foco em backend, frontend e integrações entre serviços.
Hoje meu trabalho gira em torno de aplicações full stack, com atenção especial para arquitetura, qualidade de código e evolução segura de produto.

- Backend com Go e APIs orientadas a domínio.
- Frontend com React, TypeScript e interfaces voltadas a fluxo real de uso.
- Dados e integrações com GraphQL, PostgreSQL, migrations e automações.
- Experiência com infraestrutura aplicada ao produto, incluindo Docker, EasyPanel, Vercel e rotinas de deploy.
- Interesse constante em DX, testes, padronização e simplificação de código.
